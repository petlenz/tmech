Detailed Description

template<typename _Tensor>
class detail::adjoint_wrapper< _Tensor >

Class for the computation of the adjoint of a second-order tensor.

\[ \SecondT{A}^{-1} = \frac{1}{\text{det}(\SecondT{A})}\text{adj}(\SecondT{A}) \]

TensorData type of the tensor expression.
