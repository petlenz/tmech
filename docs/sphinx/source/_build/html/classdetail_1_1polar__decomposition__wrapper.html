template<typename _Tensor>
class detail::polar_decomposition_wrapper< _Tensor >

Computes the right and left polar decomposition \(\SecondT{F} = \SecondT{R}\SecondT{U} = \SecondT{V}\SecondT{R}\) of a real second order tensor.
